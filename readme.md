## 功能说明
- 便签可以直接编辑修改标题和内容
- 便签内容增加时，便签自动变长
- 便签可以任意拖动摆放
- 内容和位置将被自动保存，不随刷新浏览器、打开关闭页面等操作变化
- 将便签拖动至垃圾桶也可以删除内容
- 删除便签时有抖动特效
- 双击某行内容或者标题，可以添加完成效果
